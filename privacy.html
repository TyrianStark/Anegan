Privacy & Data

Anegan collects minimal server and user data to function properly:

  • Logs: The bot maintains logs for analytics, media forwarding, moderation, and reminders. Each log file stores up to 50 MB of data. We keep two files for each log category; when the second file reaches 50 MB, the first file is securely deleted. Logs are only accessed when a user reports an issue or if the bot behaves unexpectedly.
  • Settings: Server-specific settings are saved in JSON or database files. This ensures that after a bot restart, power failure, or reset, your server configurations remain intact.

All data is stored securely and used solely to operate Anegan. Users in the European Union may request deletion of their personal data. Anegan only stores essential data required for its functionality.
