Privacy Policy

Effective Date: June 11, 2026

Anegan collects and processes minimal server and user data strictly required to function properly. We value your privacy and aim to be entirely transparent about how your data is handled.

1. Data We Collect & How It Is Used

  • Logs: The bot maintains temporary system logs to power core features like analytics, media forwarding, moderation, and reminders. These logs may transiently hold message content or message IDs to execute commands properly.
  • Settings: Server-specific configurations (such as custom prefixes, moderation rules, or system settings) are stored locally in JSON or database files. This ensures configurations remain intact after bot restarts, resets, or power failures.

2. Data Retention & Storage

All data is stored securely and is used solely to operate the bot. To minimize data footprint, our log files are strictly capped:

Each log category holds a maximum of two files capped at 50 MB each. When the active file reaches 50 MB, the secondary log file is automatically and securely overwritten. Logs are only ever manually accessed if a server admin reports an issue or if the bot encounters unexpected runtime errors.

3. Data Sharing & Third Parties

We do not sell, trade, or share user data with external third parties. All data transmission occurs safely through the official Discord API.

4. User Rights & Data Deletion

We believe all users should control their information. Regardless of geographic location, any server administrator or user can request the permanent deletion of their stored configurations or log footprints. To request data removal, please contact the developer via our official support Discord server (accessible through the Help tab).

5. Policy Changes

This Privacy Policy may be updated occasionally to reflect shifts in bot functionality or Discord developer requirements. Any adjustments will be posted directly to this page.
